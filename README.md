# Ternary Operator

A quick explanation:
```js
// the ternary operator
let ternary = (true) ? 'a' : 'b';

// refactors to if statement
let conditional;
if (true) {
  conditional = 'a';
} else {
  conditional = 'b';
}
```

* [js.info](http://javascript.info/ifelse#ternary-operator)
